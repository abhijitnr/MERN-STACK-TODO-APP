## Usage guide

- git clone https://github.com/abhijitnr/MERN-STACK-TODO-APP.git
- cd backend
- npm install
- add PORT and MONGODB_URL in .env file
- node index.js
- cd ../
- cd frontend
- npm install
- npm start
